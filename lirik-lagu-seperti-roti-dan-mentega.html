Sedangkan roti jenis B memerlukan 200 gram terigu dan 30 gram mentega. Jumlah maksimum roti (roti A dan roti B) yang dapat dibuat adalah


misal,
Roti A = x
Roti B = y

berdasarkan data diatas, model matematikanya adalah,
100x + 200y =< 26000
dan
20x + 30y =< 4000

cari titik maksimum dengan mencari titik potong kedua persamaan,
100x + 200y = 26000 (x 1)
20x + 30 y = 4000 (x 5)

100x + 200y = 26000
100x + 150y = 20000
eliminasi,
50y = 6000
y = 6000/50 = 120
subtitusi nilai y untuk mencari x
20x + 30(120) = 4000
20x + 3600 = 4000
20x = 400
x = 400/20 = 20

jadi, roti A = 20 dan roti b = 120

30. suatu industri rumahan pembuatan roti membuat 2 paket roti yaitu roti bantal dan roti gurih. roti bantal membutuhkan 100gr mentega dan 120gr tepung. roti gurih membutuhkan 200gr mentega dan40gr tepung. jika tersedia 7kg mentega dan 4,4kg tepung, maka model matematikan dari permasalahan tersebut adalah?


Kategori Soal : Matematika - Sistem Persamaan Linear
Kelas : X (1 SMA)
Pembahasan :
Diketahui suatu industri rumahan pembuatan roti membuat 2 paket roti, yaitu roti bantal dan roti gurih.
Roti bantal membutuhkan 100 gr mentega dan 120 gr tepung.
Roti gurih membutuhkan 200 gr mentega dan 40 gr tepung.
Tersedia 7 kg = 7000 gr mentega dan 4,4 kg = 4400 gr tepung.
Misalkan roti bantal : x dan roti gurih : y, maka model matematikanya
100x + 200y = 7000
120x + 40y = 4400

Kemudian, kedua persamaan tersebut membentuk sistem persamaan linear dua variabel sebagai berikut.
100x + 200y = 7000
⇔ x + 2y = 70
120x + 40y = 4400
⇔ 3x + y = 110
Penyelesaian sistem persamaan linear tersebut dengan menggunakan metode eliminasi dan substitusi.
Kita pilih eliminasi y, diperoleh
x + 2y = 70 |.1|
3x + y = 110 |.2|

x + 2y = 70
6x + 2y = 220
____________-
⇔ -5x = -150
⇔ x = 30
Kita substitusikan x = 30 ke persamaan 
x + 2y = 70
⇔ 2y = 70 - x
⇔ 2y = 70 - 30
⇔ 2y = 40
⇔ y = 20

Jadi, diperoleh x = 30 dan y = 20, artinya industri rumahan pembuatan roti membuat 30 buah roti bantal dan 20 buah roti gurih.

50x + 30y ≤ 2500 (diperkecil -> 5x + 3y ≤ 250)
