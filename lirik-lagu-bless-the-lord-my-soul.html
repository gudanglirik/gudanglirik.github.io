17. TOLONG :') bless your soul, amen :) |4x-80| = 75


Mapel : Matematika
Kelas : X SMA
Materi : Persamaan Nilai Mutlak

Pembahasan :
|4x - 80| = 75
(4x - 80)² = 75²
(4x - 80)² - 75² = 0
(4x - 80 + 75)(4x - 80 - 75) = 0
(4x - 5)(4x - 155) = 0
x = 5/4
x = 155/4
